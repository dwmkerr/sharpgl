SharpGL Roadmap

Below is the current roadmap for SharpGL.

SharpGL 2.0

  • Overhaul of the Core SharpGL functionality
  • Separation of WinForms code into a WinForms library
  • Support for various render context providers
  • Support for Hardware Acceleration
  • Support for Core Functionality up to OpenGL 4.2
  • Support for OpenGL extensions up to OpenGL 4.2
  • Addition of a native WPF library, without the use of WinFormsHost

SharpGL 2.1

  • Overhaul of the Scene Graph
  • Update of the roadmap
